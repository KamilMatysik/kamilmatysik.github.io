Chess in C

As part of my course in university I have to learn C, so I decided to undertake a project that would help me improve my knowledge of the language. I enjoy playing chess in my free time and felt like this would challenge would be the perfect difficulty for my skill level as I already had minor prior experience in C.

It was difficult to implement some of the logic, especially special edge-cases. I was able to implement everything other than an automatic checkmate when the king has no move as that would take knowing what every piece sees at all times and would be tough to do. The code itself can be optimised, however since this was my first time actually implementing C into a project, I was ok with it not being perfect.

It can be downloaded here and is fully playable inside of a terminal once you properly compile and execute it.

File Organiser
